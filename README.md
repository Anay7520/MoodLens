MoodLens

MoodLens is a sentiment analysis application that helps users understand the mood behind their text. It analyzes user input and classifies sentiment (positive, negative, or neutral) using transformer-based NLP models.

The project uses a Streamlit frontend, a FastAPI backend, JWT-based authentication, and MongoDB for data storage.

Features
- Sentiment analysis using transformer models
- Interactive user interface built with Streamlit
- Fast and scalable backend using FastAPI
- Secure authentication with JWT and JWTsign
- MongoDB for storing user and analysis data

Tech Stack

Frontend
- Streamlit (Python)

Backend
- FastAPI
- JWT (JSON Web Token)
- JWTsign
- Transformers for sentiment analysis

Database
- MongoDB
